# Overview

This process creates a Project within a UCL TRE. A Project is a user's work space, a browser-accessible remote desktop environment where sensitive research data can be processed. This process is owned by the TRE Environment Owner.

## 5 Safes Link

**Safe projects**: research projects are approved by data owners for the public good.

# Process
## Purpose

This process ensures that TRE team obtains all the information required by to create a Project. It ensures that Projects can only be created by Information Asset Owners/Administrators, and that the necessary Project Roles have only approved users assigned to them.

## Scope

All Projects within UCL's TRE's are within the scope of the ISMS and this process.

## Responsibilities

- Information Asset Owner: provides details, understands and owns risks related to the data and completes any other necessary compliance tasks.
- TRE Environment Team: checks and confirms that the parent Study exists and all users in the proposed Project Roles Group are Approved Researchers. Creates Projects within the Service Management Environment, creates Projects in the Technical Environment.

## Prerequisites

- To request a Project you must be either an Information Asset Owner or Information Asset Administrator. Therefore you must have already become an Approved Researcher and created a Study.

## Steps

1. Information Asset Owner/Administrator: completes request for Project to be created in an existing Study.
2. TRE Environment Team checks that:
    - the Study referenced in the Project request corresponds to an existing Study
    - the users given Egresser/Ingresser/Checker/Requester roles as well as those for whom User Desktops have been requested (collectively the Project Roles Group) are Approved Researchers
5. TRE Environment Team approves the creation of the Project (incl. the associated Project Roles Group definition) in the Service Management environment.
6. TRE Environment Team create the Project (incl. the associated Project Roles Group definition) in Technical Environment.
